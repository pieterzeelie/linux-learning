# Introduction to Linux

Linux is a free, open-source operating system based on UNIX. It is widely used for servers, desktops, embedded systems, and supercomputers due to its **security, stability, and flexibility**. A Linux system is typically distributed as a **Linux distribution (distro)**, which combines the Linux kernel with supporting libraries, utilities, and applications.
